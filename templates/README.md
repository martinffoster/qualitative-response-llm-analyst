# Canonical workbook template

The canonical Excel template is bundled with the Python package at `src/qrla/templates/qual_coding_template.xlsx`.

After installing `qrla`, create a new workbook with:

```bash
qrla init-template path/to/workbook.xlsx
```

When editing the template, update the bundled copy under `src/qrla/templates/` (the root `templates/` copy is kept for convenience when browsing the repository).
