# Data directory

Place your survey workbooks here. Workbook files are gitignored because they typically contain respondent text and other sensitive content.

To start a new question workbook, copy the bundled template:

```bash
qrla init-template data/my-survey-Q01-themes.xlsx
```

See the root `README.md` and [schemas](https://github.com/martinffoster/qualitative-response-llm-analyst/blob/main/docs/spec/schemas.md) for the expected workbook structure.
