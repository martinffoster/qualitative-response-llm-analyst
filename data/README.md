# Data directory

Place your survey workbooks here. Workbook files are gitignored because they typically contain respondent text and other sensitive content.

To start a new question workbook, copy the template:

```bash
cp templates/qual_coding_template.xlsx data/my-survey-Q01-themes.xlsx
```

See the root `README.md` and `docs/spec/schemas.md` for the expected workbook structure.
